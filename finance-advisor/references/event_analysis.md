# 政治/宏观事件分析框架

## 分析原则

1. **信息来源多元化**：同时查阅 Bloomberg、Reuters、FT、WSJ、SCMP 等主流媒体
2. **时间敏感性**：优先关注近 3 天事件，重要事件延伸至近 2 周
3. **传导路径清晰**：事件 → 情绪面 → 资金面 → 基本面 → 资产价格
4. **不做预测只做应对**：分析不同情境下各类资产的潜在表现，给出概率性判断

## 事件分析 SOP

### Step 1: 事件识别
```
来源：web_search 搜索 "重大财经事件 YYYY-MM-DD"
优先：Bloomberg/Reuters/FT 头条
筛选：近3天核心事件，按影响范围排序
```

### Step 2: 情境分析
```
每个事件分析三个情境：
- 基准情境（60%概率）：符合市场共识
- 乐观情境（25%概率）：超出预期利好
- 悲观情境（15%概率）：低于预期或黑天鹅

对每个情境给出：
1. 短期（1-5天）价格影响
2. 中期（1-4周）趋势影响
3. 建议操作（买入/持有/卖出/观察）
```

### Step 3: 持仓影响评估
```
将事件情境映射到用户持仓：
✅ 看多信号：考虑加仓或维持
⚠️ 不确定：保持观望，不追涨
❌ 看空信号：考虑减仓或止损
```

## 快速情绪判断框架

**黄金>$2050**：全球避险情绪强，资金流入金条/ETF/矿业股
**VIX>20**：市场恐慌，波动率高，适度减仓高波动品种
**美元指数>104**：美元强势，新兴市场/港股承压
**10年美债收益率>4.5%**：无风险利率上升，成长股估值承压
**USDCNH>7.30**：人民币贬值预期，A/H股外资流出
